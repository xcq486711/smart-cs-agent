## 1. 核心收获

### 语言模型演进
- **N-gram**: 用"数数"算概率。稀疏性+无语义理解→失败
- **词嵌入**: 把词变成向量。king - man + woman ≈ queen。向量空间能"算语义"
- **RNN/LSTM**: 给网络加记忆。梯度消失→长序列学不到开头的信息
- **Transformer**: 抛弃串行，Self-Attention 让每个词直接看所有词。并行+长距离依赖一站式解决

### Transformer 六组件（代码一一对应）

| 组件 | 一句话功能 | 关键代码 |
|---|---|---|
| Self-Attention | 每个词看所有词，Query×Key→权重×Value | `QK^T/√dk` → Softmax → ×V |
| Multi-Head | 多个角度同时看 | 8 个头并行跑 Attention，拼接输出 |
| Positional Encoding | 知道谁在前谁在后 | sin/cos 生成位置向量，加到词嵌入上 |
| FFN | 知识存储+非线性变换 | 512→2048→512，中间激活 |
| Residual+LayerNorm | 梯度不消失+数值不炸 | x + F(x) → LayerNorm |
| Decoder-Only | 单向生成，预测下一个词 | Masked Self-Attn，无 Cross-Attn |

### Prompt 工程 = 用自然语言编程
- **Few-shot**: 给示例 → 模型在上下文中"学会"你的格式
- **Chain-of-Thought**: "让我们一步一步推理" → 推理过程的 token 成为后续推理的上下文
- **Thought 的本质**: 不只是调试信息，是模型自己给自己写的草稿

### 幻觉 = 机制的内在特性，不是 bug
- 事实性幻觉 → RAG 注入最新信息覆盖 FFN 旧知识
- 逻辑性幻觉 → Chain-of-Thought 显式推理
- 格式性幻觉 → Few-shot 示例 + 正则兜底
- 归因性幻觉 → Prompt 约束"只用我给的资料"
