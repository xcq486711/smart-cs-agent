## 1. 核心主线

**智能体发展 = 问题驱动的演进，每一代都在解决上一代的痛点。**

```
1950s 符号主义 → 1980s 联结主义 → 1990s 强化学习 → 2020s LLM Agent
 (规则引擎)      (神经网络)       (试错学习)       (三者融合)
```

---

## 2. 各阶段要点

### 符号主义（§2.1）
- 核心信念：智能 = 符号操作 + 逻辑推理
- 代表：MYCIN（600条规则诊断血液感染）、SHRDLU（积木世界自然语言交互）
- 致命伤：知识获取瓶颈、框架问题、脆弱性
- **遗产**：推理链思维、知识库概念、可解释性

### ELIZA 实践（§2.2）
- 1966年，不到100行代码，正则匹配+模板替换
- 完全不理解语义——"I am not happy" → 回复"How long have you been not happy?"
- **ELIZA 效应**：用户会主动给机器"脑补"智能

### 心智社会（§2.3）
- 明斯基的核心洞见：智能不从单一完美系统产生，从大量简单Agent协作中涌现
- 去中心化、涌现式计算、Agent社会性
- **对现代 MAS 的启发**：多Agent协作 > 单一巨型Agent

### 学习范式（§2.4）
- 联结主义：知识分布式存储在神经网络权重中，从数据中学习
- 强化学习：通过"试错+奖励信号"学习序贯决策（AlphaGo）
- 预训练+LLM：互联网规模的数据 → 世界知识 + 涌现能力

### LLM Agent（§2.4.4）
- **LLM Agent = 符号主义 + 联结主义 + 强化学习的融合体**
- 架构：感知模块 → 规划模块 → LLM → 执行模块 → 工具调用 → 观察 → 循环
